    Distributed Remote Boot Linux With Kali

    A project designed for my alma mater. The student run cyber security organization was struggling to get students to join and particpate in competitions, we found the issue to be that students could not jump into the environment and begin learning due to needing a lot of setup. This project is supposed to solve that issue by allowing a new students to setup their machine to PXE boot and have their machine do the computation, but save all files to the remote server. This will allow for minimal client side setup.
